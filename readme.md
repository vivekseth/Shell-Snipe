# Shell Sniper #

## Usage ##

takes 4 paramaters

1. email
2. subject
3. course
4. section

## Examples ##

`$ sh ./snipe.sh <email> <subject> <cource> <section>`
`$ sh ./snipe.sh vivekseth.m@gmail.com 198 352 01`
